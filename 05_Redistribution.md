# 5. Redistribution

Redistribution describes how load reorganizes across the field in response to changing conditions.

Where propagation spreads consequence, redistribution adjusts where the load is carried.

Redistribution is not about spread.
It is about rebalancing.

Pressure accumulates in one region.
The field responds.
Adjacent regions adjust.
Responsibility shifts.

A new configuration emerges.

Redistribution preserves integrity by preventing any single region from bearing more load than it can sustain.

The field does not return to a previous state.
It adapts.
Load is shared differently.
Responsiveness remains possible.

Integrity remains visible through the reorganization of support.

Redistribution is the morphology through which the field remains workable under pressure.

It is the second core expression of circulation.
